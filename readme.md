10-Day JavaScript  

Day 1: Fundamentals & Syntax
Topics to Cover:

Variables (var, let, const) and scope
Data types (primitives: string, number, boolean, null, undefined, symbol, bigint)
Type conversion and coercion
Operators (arithmetic, comparison, logical, assignment)
Comments and code structure
Template literals and string methods
Basic input/output (console.log, alert, prompt)

Practice Questions:

Create a temperature converter (Celsius to Fahrenheit and vice versa)
Build a simple calculator that takes two numbers and an operator
Write a program to check if a number is even/odd and positive/negative
Create a age calculator that calculates age from birth year

Mini Project: Personal Info Card Generator - Create a program that takes user inputs (name, age, city, occupation) and displays a formatted info card using template literals.

Day 2: Control Flow & Loops
Topics to Cover:

If-else statements and ternary operator
Switch statements
For loops (traditional, for...in, for...of)
While and do-while loops
Break and continue statements
Truthy and falsy values
Short-circuit evaluation

Practice: Create programs with decision-making logic and iterations

Day 3: Functions
Topics to Cover:

Function declarations vs expressions
Arrow functions
Parameters and arguments (default, rest, spread)
Return statements
Function scope and closures
IIFE (Immediately Invoked Function Expressions)
Callback functions
Higher-order functions

Practice: Build utility functions and work with callbacks

Day 4: Arrays & Array Methods
Topics to Cover:

Creating and accessing arrays
Array methods: push, pop, shift, unshift
Iteration methods: forEach, map, filter, reduce
Finding: find, findIndex, indexOf, includes
Sorting and reversing: sort, reverse
Modifying: slice, splice, concat
Other methods: join, split, every, some
Array destructuring

Practice: Solve array manipulation problems

Topics to Cover:

Object literals and properties
Accessing properties (dot vs bracket notation)
Object methods

Prototypes and prototype chain
Classes (ES6)
Inheritance and extends
Static methods
Getters and setters

Practice: Create object-based programs and class hierarchies

Topics to Cover:

Selecting elements (getElementById, querySelector, etc.)

Changing styles and classes
Event listeners and event object

Event delegation
Form handling and validation

Practice: Build interactive web pages

Topics to Cover:

Synchronous vs asynchronous code
setTimeout and setInterval
